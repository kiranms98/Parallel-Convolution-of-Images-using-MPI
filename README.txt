1. The following code can be run using mpirun -n "processes" python3 grid.py
2. Please make sure thay Python Pillow, mpi4py, numpy libraries have been installed
3. The output of the code is a transformed image which can be found in the same directory as the code
4. Please make sure that the correct address is specified to the source image
5. Please make sure that the image format specified for the output image is same as the input image
6. The grid.py works only when the number of specified processes is a perfect square
